# Blog Review Editorial

Use this checklist:

1. Read the full post before editing.
2. Fix typos, spelling, punctuation, possessives, and obvious grammar issues immediately.
3. Do not change tone, message, argument, structure, emphasis, or voice without approval.
4. Do not tighten ideas, remove lines, add examples, add lists, or rewrite sections without approval.
5. If a draft feels too abstract, propose a short list instead of adding one directly.
6. Look for natural opportunities for inline internal links.
7. Consider backlinks from related posts when the overlap is real.
8. Treat internal links and backlinks as editorial changes that need approval before applying.
9. Prefer inline links only. Do not add callouts, related-post blocks, or CTA language.
10. When proposing edits, give concrete line-specific suggestions the author can approve quickly.
11. When reporting back, separate edits already applied from edits still awaiting approval.
